Chem-Cat:


Chem-Cat is a mobile app for learning organic chemistry, available on both Google Play and The App Store. It utilizes Kivy, an open source Python library for developing mobile applications. The app features over 75 pages of illustrated explanations detailing key concepts of organic chemistry, over 900 questions, and 10 sets of step-by-step flashcards for commonly asked reaction mechanisms.


This project honed my ability to problem solve. Figuring out the Kivy UI, how it interacted with my Python code, and running test after test to get things to appear/work the way I wanted them to was a great learning experience. In addition, getting my app (written in Python and Kivy Language) to compile into an APK/iOS package using VM environments - as well as programs like HomeBrew, Buildozer, and more advanced functions of the command line - was another point of growth for me.

HackerRank/Project Euler:


HackerRank and Project Euler are both problem solving websites. The general theme of both is that there is an obvious brute force solution, but finding an optimized runtime solution is necessary to complete each challenge. These sites are one of the ways that I test my understanding of concepts.


I enjoy these sites because they let me test my ability to create efficient solutions to problems in terms of time and space complexity (admittedly with more emphasis on efficient runtime). While a lot of my current projects tend to be more UI related, in my programming journey I have definitely spent the bulk of my time understanding/ mastering topics like data structures, algorithms, OOP, etc., and how to use them to produce efficient Big-O runtimes for data manipulation. These sites provided me with an environment to practice and hone these skills, complete with diverse problems and test cases.
