Personal Projects

Chem-Cat:

Chem-Cat is a mobile app for learning organic chemistry, available on both Google Play and The App Store. It utilizes Kivy, an open source Python library for developing mobile applications. The app features over 75 pages of illustrated explanations detailing key concepts of organic chemistry, over 900 questions, and 10 sets of step-by-step flashcards for reaction mechanisms.

This project honed my ability to problem solve. Figuring out the Kivy UI, how it interacted with my Python code, and running test after test to get things to appear/work the way I wanted them to was a great learning experience. In addition, getting my app (written in Python and Kivy Language) to compile into an APK/iOS package using VM environments - as well as programs like HomeBrew, Buildozer, and more advanced functions of the command line - was another point of growth for me.

Bookmark

Bookmark is a chrome extension that keeps track of the scroll location of points the user specifies, allowing them to be returned to simply by clicking the bookmark made by the user for that location. Bookmark allows you to store up to 10 bookmarks per website for up to four websites, allowing different sections of the long webpage to be jumped between with ease and efficiency, instead of having to search for them.

Due to the COIVD-19 pandemic I was unable to partake in weekend/post-work activities that I would normally do so I needed something to fill the void. I had been interested in chrome extensions, and so making one seemed like a good way to fill new time I had on my hands. Learning the extension setup and Chrome API, as well as getting to test and problem shoot the abilities the API provided, was a fun and educational experience and I was satisfied with the final product I produced.

HackerRank/Project Euler:

HackerRank and Project Euler are both problem solving websites. The general theme of both is that there is an obvious brute force solution, but finding an optimized runtime solution is necessary to complete each challenge. These sites are one of the ways that I test my understanding of concepts.

I enjoy these sites because they allowed me to test my ability to create efficient, streamlined solutions by testing my knowlege of data structures, algorithms, OOP, etc., and how to use them to produce efficient Big-O runtimes for data manipulation. These sites provided me with an environment to practice and hone these skills, complete with diverse problems and test cases. I am currently in the top 3% for number of problems solved on Project Euler, and have numerous gold medal achievments on HackerRank.

Tryout:

Tryout is a mobile app whose code is available on my Github. I made it using Kivy, just as I did with Chem-Cat. The app’s inspiration came from my experience coaching high school ultimate frisbee, I wanted to make an app to track playing time and ensure equal opportunities to see the field. Tryout also has options for football, soccer, rugby, field hockey, and basketball. The app runs fine on a desktop using Kivy, but is yet to be released on The App Store/Google Play.

Personal Website:

Many job applications asked for a personal website, and I didn't have one. So, I built my own (mobile and desktop compatible) from scratch using HTML/CSS/JavaScript.
